November 4, 2025

v0.2.6: Production Security & Infrastructure

Production readiness with enterprise-grade TLS/HTTPS and mutual TLS support, built-in tunneling for local development, and completed SDK implementation.

TLS/HTTPS and mTLS Support

Comprehensive TLS/HTTPS with configurable certificate and key file paths for all HTTP endpoints. Optional mutual TLS (mTLS) adds client certificate authentication with configurable client CA chains, ensuring only authorized clients connect.

Built-in Tunneling Service

Expose local MockForge instances publicly via mockforge serve --tunnel. Automatic public URL generation for webhook testing, OAuth callback URLs, mobile app development, and remote team collaboration.

SDK Implementation Complete

Phase 1 and 2 of the MockForge SDK are complete: production-ready client generators for React, Vue, Svelte, and Angular with comprehensive documentation, real-world examples, and full TypeScript support.

Infrastructure Improvements

Unified versioning across all workspace crates, fixed version mismatches in UI and plugin loader. Added mockforge-tcp and mockforge-test to the publish script. Fixed Axum 0.8 compatibility in proxy server and MQTT error types for async/await. Improved module-level documentation consistency.
