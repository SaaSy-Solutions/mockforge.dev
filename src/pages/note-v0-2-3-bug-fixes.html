October 31, 2025

v0.2.3: Bug Fixes & Stability Improvements

A stability release addressing critical bugs in OpenAPI example extraction, request body parameter generation, and React client generator templates.

OpenAPI Example Extraction Fix

Explicit examples from OpenAPI schemas and properties are now correctly prioritized over generated mock data. Previously, specified examples at the schema or property level were sometimes ignored in favor of random generated values.

Request Body Parameter Generation Fix

The React client generator now correctly generates data parameters for POST, PUT, PATCH, and DELETE operations with proper JSON serialization via JSON.stringify(data).

Template Logic Fixes

Handlebars template logic for request body type generation now correctly handles complex schemas and nested types. React hook useCallback dependency arrays are properly formatted, preventing React warnings.

Test Coverage

Comprehensive test coverage added for request body parameter scenarios (all HTTP methods), complex nested schemas, array request bodies, multiple Content-Type support, and example extraction priority.
