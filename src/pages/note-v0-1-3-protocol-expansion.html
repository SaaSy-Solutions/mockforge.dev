October 22, 2025

v0.1.3: Protocol Expansion & Testing Tools

A significant milestone expanding protocol support and introducing powerful testing capabilities including native GraphQL handlers, programmable WebSockets, HTTP scenario switching, and capture scrubbing.

Native GraphQL Operation Handlers

First-class GraphQL support with schema-aware query and mutation handling, operation validation, field-level response customization, subscription support, fragment handling, and variable substitution.

Programmable WebSocket Handlers

Real-time testing via custom WebSocket handlers with message pattern matching, stateful connections, custom handler logic, and replay support for simulating chat applications and live data feeds.

HTTP Scenario Switching

Dynamically switch between multiple OpenAPI response examples at runtime via admin UI or API, making it straightforward to test different API states (active, suspended, deleted) without changing configuration.

Capture Scrubbing & Deterministic Replay

Automatic PII removal from recorded traffic (emails, phone numbers, SSNs), custom scrubbing rules, and deterministic replay with seed management for consistent, CI/CD-ready tests.

Unified Config & Profiles

Multi-format config support (YAML, JSON, TOML), profile-based configuration for dev/staging/prod, environment variable overrides, and schema validation.

Testing Tools

New mockforge-test crate for integration testing, plus Kubernetes-style health endpoints (/healthz, /ready, /live).
